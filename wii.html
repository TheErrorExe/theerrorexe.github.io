Installation

Console

Download the WiiMart WAD and the patched IOS WAD for your console type (Wii/vWii). We recommend YAWMME for installing WADs.

  • vWii is the Wii U’s Wii Mode
  • If you get an error like -1022 or -1029, delete the existing title first.

Dolphin

For those with Wii NAND backups:

  • Install both WADs in Dolphin via Tools > Install WAD.
  • Go to the Wii Menu via Tools > Load Wii System Menu 4.3X.

For those without Wii NAND backups:

So far, getting the shop on a blank NAND has been unsuccessful.
